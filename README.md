# RealVisualLight – Motore di visualizzazione ottimizzato per FreeCAD

RealVisualLight è un motore di visualizzazione ottimizzato per FreeCAD, progettato per migliorare la fluidità, la chiarezza e la reattività della vista 3D. Applica automaticamente impostazioni grafiche leggere e personalizzabili ogni volta che si apre un documento o si cambia ambiente di lavoro, offrendo un’esperienza più pulita e scorrevole senza modificare gli strumenti di modellazione.

## Funzionalità principali
- Attivazione automatica all’apertura dei documenti
- Attivazione automatica al cambio di Workbench
- Impostazioni grafiche personalizzabili tramite RVL_Settings.py
- Navigazione 3D più fluida
- Visualizzazione più chiara e pulita
- Nessuna modifica agli strumenti di modellazione
- Progetto completamente gratuito e open‑source

## Installazione
Copiare la cartella `RealVisualLight` in:

`C:\Users\<nome>\AppData\Roaming\FreeCAD\Mod\`

Oppure aggiungere questo repository come sorgente personalizzata nell’Addon Manager.

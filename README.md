# RealVisualLight – Optimized Visualization Engine for FreeCAD

RealVisualLight is an optimized visualization engine for FreeCAD, designed to improve the smoothness, clarity, and responsiveness of the 3D view. It automatically applies lightweight and customizable display settings whenever a document is opened or a workbench is switched, delivering a cleaner and more fluid experience without altering any modeling tools.

## Main Features
- Automatic activation when opening documents
- Automatic activation when switching workbenches
- Customizable visual settings via RVL_Settings.py
- Smoother 3D navigation
- Cleaner and clearer viewport
- Does not modify any modeling tools
- Fully free and open‑source project

## Installation
Copy the `RealVisualLight` folder into:

`C:\Users\<yourname>\AppData\Roaming\FreeCAD\Mod\`

Or add this repository as a custom source in the Addon Manager.
